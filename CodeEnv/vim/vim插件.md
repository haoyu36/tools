## 插件使用


ctrlpvim/ctrlp.vim 文件搜索

```
ctrl + j/k 进行上下选择

ctrl + x 在当前窗口水平分屏打开文件

ctrl + v 同上, 垂直分屏

ctrl + t 在tab中打开
```
